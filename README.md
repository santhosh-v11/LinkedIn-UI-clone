# LinkedIn UI Clone using Tailwind CSS only

## Description
This project is part of my internship work to learn and improve skills in Tailwind CSS. The main task was to clone the LinkedIn UI using Tailwind,focusing on layout,responsiveness,and component styling to get hands-on experience.

## Features
- Grid-based layout for a clean, responsive UI
- Built entirely with Tailwind CSS utility classes
- Responsive breakpoints for mobile, tablet, and desktop

## Tech Stack
- HTML5
- CSS3 
- TailwindCSS
- npm and Vite
- versal for deployment
